# Sales Data Analysis

## 📌 Description
This project analyzes sales data using Python to extract meaningful insights and visualize trends.

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib

## 📊 Features
- Data cleaning and preprocessing
- Sales analysis by category
- Sales analysis by region
- Data visualization using bar charts

## 🚀 How to Run
1. Install required libraries:
   pip install pandas matplotlib

2. Run the script:
   python main.py

## 📈 Output
- Displays total sales by category and region
- Generates bar charts for better understanding
